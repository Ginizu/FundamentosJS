Estrutura de dados clássicas

Formas mais oprganizadas para organização de dados de forma mais eficiênte

  • Lista Ligada: coleção ordenada de valores onde cada elemento aponta para o próximo.
  • Array: Coleção de elementos que podem ser idenficados por um índice.
  • Fila(Queue): Coleção de elementos onde o primeiro a entrar é o primeiro a sair.
  • Pilha(Stack): Direfente do Queue, aqui é o o ultimo ao entrar é o primeiro a sair.
  • Árvore(Tree): Conjuntos de elementos entre nos, se comporta como uma arvore genealógicas e a base de hieraquias.
